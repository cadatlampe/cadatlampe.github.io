# Mellouki Cadat-Lampe — GitHub Pages site

Bestanden:
- `index.html`
- `style.css`
- `script.js`

## Publiceren via GitHub Pages
1. Maak op GitHub een nieuwe repository.
2. Upload deze bestanden in de root van de repository.
3. Ga naar **Settings → Pages**.
4. Kies **Deploy from a branch**.
5. Kies branch **main** en folder **/ (root)**.
6. Wacht tot GitHub de site heeft gepubliceerd.

### URL
- Voor een **gebruikerssite**: repositorynaam `jouwgebruikersnaam.github.io`
- Voor een **projectsite**: willekeurige repositorynaam; URL wordt dan `https://jouwgebruikersnaam.github.io/repositorynaam/`

## Opmerking
De site is volledig statisch en gebruikt geen build tooling.
